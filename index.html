OPENLENS

OpenLens helps students and developers discover beginner-friendly GitHub repositories by analyzing project activity, issues, and documentation.

What is OpenLens?

OpenLens is a web-based analytical platform designed to assist students and aspiring developers in identifying suitable open-source repositories for contribution. It evaluates repository activity, issue complexity, documentation quality, and contributor engagement to determine contribution readiness.

Features

Contribution Readiness Scoring

Automatically evaluates repositories for suitability.

Beginner-Friendly Discovery

Find repositories aligned with your skills.

Issue & Activity Analysis

Analyzes issues and contributor activity.

API-Driven Insights

Uses GitHub APIs for real-time analytics.
