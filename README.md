# OpenLens
AI-Based GitHub Contribution Readiness & Repository Discovery System

OpenLens is a developer tool designed to help programmers discover open-source repositories and evaluate whether they are suitable for contribution.

Many developers struggle to find beginner-friendly open-source projects. OpenLens analyzes repository data and provides insights to determine contribution readiness and project complexity.

---

# 🚀 Features

### Repository Discovery
Search and explore GitHub repositories based on keywords and technologies.

### Contribution Readiness Analysis
Evaluate repositories to determine how suitable they are for new contributors.

### Difficulty Classification
Repositories are categorized into:
- Beginner Friendly
- Intermediate
- Advanced

### Developer Analytics Dashboard
Interactive dashboard showing:

- Total Analyses
- Contribution Readiness Score
- Saved Repositories
- Recent Analyses

### Repository Insights
Provides insights such as:

- Project activity
- Contribution frequency
- Repository maintenance status
- Issue and pull request statistics

---

# 🧠 Motivation

Open-source contribution is one of the best ways for developers to gain real-world experience. However, finding beginner-friendly repositories is difficult.

OpenLens aims to solve this by analyzing GitHub repositories and helping developers identify projects where they can successfully contribute.
